# ⚡ IoT-Driven Electrical Fault Monitoring & Predictive Maintenance System

## 🔹 Overview
This project presents an end-to-end solution for electrical fault monitoring and predictive maintenance using IoT data, Machine Learning, and Power BI. It combines data visualization with predictive analytics to detect, analyze, and forecast electrical faults.

---

## 🔹 Dataset
- Custom dataset created for electrical fault analysis
- Includes parameters like voltage, current, temperature, and fault indicators
- Simulates real-world IoT sensor data

---

## 🔹 Machine Learning Model
- Algorithm Used: Random Forest
- Used for predicting potential electrical faults
- Model trained on historical sensor data
- Provides classification of fault risk levels (High, Medium, Low)

---

## 🔹 Dashboards Included

### 1. Electrical Fault Monitoring Dashboard
- Real-time visualization of fault data
- System health overview

### 2. Electrical Fault Monitoring and Trend Analysis
- Historical trend analysis
- Pattern identification

### 3. Predictive Electrical Fault Monitoring Dashboard
- Displays ML-based predictions
- Shows risk levels and probabilities

### 4. Advanced Predictive Dashboard
- KPI comparisons
- Detailed fault insights

### 5. Future Scope & IoT-Driven Predictive Maintenance
- Focus on real-time IoT integration
- Automation possibilities
